# Bluestock IPO Web Application

A full-stack IPO tracking platform with REST API for Bluestock Fintech.

## Tech Stack
- **Backend**: Node.js (Express)
- **Frontend**: React.js
- **Database**: PostgreSQL

## Setup
1. Clone the repo
2. Run `npm install`
3. Configure `.env` (see `.env.example`)
4. Seed the database: `node scripts/seedData.js`
